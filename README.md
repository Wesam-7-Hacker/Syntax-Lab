منصة 𝙒𝙚𝙨𝙖𝙢 𝙃𝙖𝙘𝙠𝙚𝙧 𝙤𝙛 𝙄𝙨𝙡𝙖𝙢
هي منصة متعددة الوظائف لاختبار الاختراق (Penetration Testing)، تم تطويرها باستخدام لغة بايثون ومكتبة CustomTkinter، وتتميز بواجهة مستخدم حديثة. صُممت هذه الأداة لإدارة كل من عمليات الاستطلاع السلبي (recon) ومحاكاة الهجمات النشطة من مركز تحكم واحد.
​⚠️ تحذير: هذه الأداة مخصصة لأغراض التعليم واختبار الاختراق الأخلاقي فقط. قد يكون استخدامها على الأنظمة غير المصرح بها أمرًا غير قانوني. المستخدم يتحمل المسؤولية الكاملة عن جميع تصرفاته.
​🚀 الميزات
تم تجهيز 𝙒𝙚𝙨𝙖𝙢 𝙃𝙖𝙘𝙠𝙚𝙧 𝙤𝙛 𝙄𝙨𝙡𝙖𝙢
 بـ 11 وحدة أمنية حرجّة ومختلفة:
​🔍 الاستطلاع السلبي (Passive Recon)
Super Fast Port Scan: يكتشف المنافذ المفتوحة على عنوان الـ IP/Host المستهدف ويقرأ لافتات الخدمة (مثل SSH، HTTP، وغيرها).
​Reverse IP Lookup: يحدد المواقع الإلكترونية الأخرى المستضافة على نفس عنوان الـ IP.
​Subdomain Finder: يسرد النطاقات الفرعية (subdomains) المرتبطة بالنطاق المستهدف.
​WAF Detector: يتحقق مما إذا كان الموقع محميًا خلف جدران حماية مثل Cloudflare أو Sucuri أو ModSecurity.
​JS Secret Leaks: يفحص ملفات الجافا سكريبت للبحث عن مفاتيح الـ API الداخلية وكلمات المرور المخفية.
​⚔️ الهجوم النشط (Active Attack)
Auto-Crawler Pro: يتنقل في الموقع تلقائيًا للعثور على جميع الروابط ومعلمات روابط الـ URL.
​Mass Shell Hunter: يبحث عن ثغرات رفع الملفات الحرجّة (upload) في الخادم.
​SQLi Auto-Tester: يرسل حمولات (payloads) تلقائية للكشف عن ثغرات حقن الـ SQL.
​Admin Panel Finder: يحاول العثور على صفحة إدارة الموقع (admin page) باستخدام قائمة مسارات واسعة.
​🛠️ أدوات النظام (Utils)
Exploit Architect (AI): يقترح رموز ثغرات الـ CVE وطرق الاستغلال المتوافقة مع أرقام الإصدارات بالاعتماد على قاعدة بياناته الداخلية.
​Generate Report: يحفظ نتائج الفحص في ملف بصيغة .txt مع طابع زمني.
